Installation¶

Requirements¶

db-load-generator requires following dependencies to be present in the environment.

  • Python >= 3.9, < 4.0

  • Java >= 8

  • JDBC driver for the database

Installing from PyPI¶

db-load-generator can both be installed as a system wide package and as a dependency in a python project.

pip install db-load-generator
